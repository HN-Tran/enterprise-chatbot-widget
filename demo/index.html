Enterprise Chatbot Widget

Demo-Seite für den Dokument-Assistenten

Willkommen

Dies ist eine Demo-Seite für das Enterprise Chatbot Widget. Klicken Sie auf den blauen Button unten rechts, um den Assistenten zu öffnen.

Der Chatbot verbindet sich mit dem Enterprise RAG System und kann Fragen zu Ihren Dokumenten beantworten.

Funktionen

  • Streaming-Antworten - Antworten werden in Echtzeit angezeigt
  • Quellenangaben - Jede Antwort enthält Referenzen zu den Originaldokumenten
  • Session-Speicherung - Der Chatverlauf bleibt beim Seitenwechsel erhalten
  • Kopieren - Antworten können mit einem Klick kopiert werden
  • Feedback - Bewerten Sie Antworten mit Daumen hoch/runter

Integration

So integrieren Sie das Widget in Ihre Seite:

<script src="chatbot-widget.js" data-auto-init="true"></script>

Oder mit mehr Kontrolle:

<script src="chatbot-widget.js"></script>
<script>
  EnterpriseChat.init({
    position: 'bottom-right',
    theme: { primaryColor: '#1a73e8' },
    labels: { title: 'Dokument-Assistent' }
  });
</script>
Hinweis: Die API-URL wird beim Build in das Widget eingebettet. Build mit: VITE_API_URL=https://ihr-server.de npm run build

Test ohne Backend

Um das Widget ohne ein laufendes RAG-Backend zu testen, können Sie die Browser-Konsole öffnen, um Netzwerkfehler zu sehen. Das UI funktioniert trotzdem und zeigt eine Fehlermeldung an.
